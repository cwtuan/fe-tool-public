# React Todo Demo App

A simple React todo list application built with Vite.

## Features

- Add new todos
- Mark todos as completed (toggle)
- Delete todos

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173` (or the port shown in the terminal).

## Usage

- Enter a todo item in the input field and click "Add" to add it to the list.
- Click "Toggle" to mark a todo as completed or incomplete.
- Click "Delete" to remove a todo from the list.

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
